# CardPin Journey Design Canvas

This directory contains design artifacts for analyzing and improving the user journey in CardPin.

## Files

- `cardpin-journey-ccanvas.html` - A design canvas showing three different journey perspectives:
  1. **Data-Focused Design** - Emphasizes transparency and full information display
  2. **Action-Focused Design** - Prioritizes decision speed and clear action paths  
  3. **Storytelling Design** - Uses narrative elements to guide understanding

## Purpose

The design canvas helps visualize how different approaches to user journey design affect:
- Information hierarchy and presentation
- Decision-making support 
- Visual complexity and cognitive load
- Integration of reward information with wallet management
- Mobile responsiveness considerations

## How to Use

1. Open `cardpin-journey-ccanvas.html` in a web browser
2. View each variation side-by-side to compare approaches
3. The canvas demonstrates how different design philosophies impact the user's journey through making card selection decisions
4. Each variation shows the core components of CardPin:
   - Country selector and merchant context 
   - Recommendation display with reward information
   - Spend amount controls
   - Wallet management interface

## Design Principles

1. **Data-Focused**: Provides comprehensive data for informed decision-making, including all calculation details and alternative options.

2. **Action-Focused**: Simplifies the interface to enable quick decisions by reducing visual complexity while maintaining essential functionality.

3. **Storytelling**: Uses narrative elements to guide users through their decision process and help them understand why certain cards are recommended.
